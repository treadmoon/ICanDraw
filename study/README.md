# ICanDraw 教学案例：用 AI 从零构建一个完整产品

> 本教学材料以 ICanDraw 项目为真实案例，完整还原"从一句话想法到可运行产品"的全过程。
> 适合想了解 AI 编程方法论的开发者、产品经理、技术管理者。

---

## 教学目标

学完本案例，你将掌握：

1. 如何用结构化流程（Spec-Kit）驱动 AI 编程，而不是"对着 AI 瞎聊"
2. 每个阶段的输入、输出、关键决策点
3. AI 编程中"人"的角色——判断力比编码能力更重要
4. 文档如何充当 AI 的"外置记忆"，解决上下文丢失问题
5. 实际踩过的坑和修复过程

## 项目概况

- 产品名称：ICanDraw — AI 数据叙事智能画布
- 技术栈：Next.js 16 + Excalidraw + ECharts + Zustand + 火山方舟 API
- 开发方式：核心代码全部由 AI 生成，人负责需求定义、架构决策、质量把关
- 工具链：Kiro CLI + Spec-Kit v0.4.2

## 前置知识

- 基本的 Web 开发概念（前端/后端/API）
- 了解 React 和 TypeScript 更佳，但非必须
- 不需要有 AI 编程经验——这正是本教程要教的
